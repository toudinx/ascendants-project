# Ascendants Project

## Quick Start

Install dependencies:
```bash
npm i
```

Run the app:
```bash
npm start
```

Run tests:
```bash
npm test
```

Run lint:
```bash
npm run lint
```

## Folder Overview

- `src/app/core`: core services, models, state, and shared logic.
- `src/app/content`: content definitions and balance/config data.
- `src/app/features`: feature modules and pages.
